# Asenxo-Dashboard

https://asenxo-dashboard-s9wdyspzmdf3ackq7dddhm.streamlit.app/

### ASENXO is a web-based analytics dashboard built with Streamlit that supports the DOST SETUP 4.0 iFund Program in monitoring and evaluating MSME beneficiaries across Region VI. It provides program administrators and technical officers with data-driven insights through two core analytics components.
